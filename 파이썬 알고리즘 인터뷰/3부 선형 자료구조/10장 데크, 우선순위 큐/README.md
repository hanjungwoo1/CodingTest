# 데크, 우선순위 큐

## 데크
데크(Deque) : 더블 엔디드 큐(Double-Ended Queue)의 줄임말로,
글자 그대로 양쪽 끝을 모두 추출할 수 있는, 큐를 일반화한 형태의 추상 자료형(ADT)이다.
